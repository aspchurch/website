**Priority:** High
**Estimated Timeline:** 1-3 months

### Recommended Actions:

1. **Keyword Research**
   - Identify primary target keywords:
     - "presbyterian church carlsbad"
     - "pca church north county"
     - "reformed church carlsbad"
     - "church carlsbad sunday evening"
     - "bible church north county san diego"

2. **Content Expansion**
   - **FAQ Page:** Answer common questions
     - "What is a Presbyterian church?"
     - "What does PCA stand for?"
     - "What to expect at your first visit"
     - "What is Reformed theology?"
     - "Do you have childcare?"

   - **Expand Existing Pages:**
     - Add more detail to beliefs page (SEO-rich content)
     - Expand about page with church history
     - Add more detail about service structure

   - **New Content:**
     - Ministries/Small Groups page
     - Events calendar/page

3. **Blog/Sermon Section** (High Impact)

   - Benefits:

     - More keyword opportunities
     - Increases site authority

4. **Structured Data (Schema.org)**

## Common SEO Issues to Avoid

1. **Keyword Stuffing:** Don't overuse keywords unnaturally
2. **Duplicate Content:** Keep all page content unique
3. **Broken Links:** Regular audits to fix 404s
4. **Slow Loading:** Keep page speed optimized
5. **Mobile Issues:** Ensure mobile-first design
6. **Missing Alt Text:** Add to all images
7. **Thin Content:** Pages should have 300+ words minimum
8. **Inconsistent NAP:** Keep contact info identical everywhere
